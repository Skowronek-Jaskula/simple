Quiz after Day 1.

1. How does inheritance work in CSS?

HTML: 
    <div class="parent">
      <p class="child">What is my color?</p>
    </div>
CSS: 
    .parent {
       color: blue;
    }

2. Which of the following methods is incorrect for defining styles in CSS?

HTML inline styling: 
    <p style="color: red;">Text</p>
CSS nested styling: 
    .nested .nested__item {
       font-size: 14px;
    }
JS dynamic styling: 
    document.getElementById("el").style.color = "red"

3. What is specificity of selectors in CSS?

HTML: 
    <p id="myId" class="my-class">What is my color?</p>
CSS: 
    p {
       color: blue;
    }
            
    .my-class {
        color: red;
    }
            
    #myId {
       color: green;
    }

4. How does the order of styles change in CSS?

CSS: 
    p {
      color: red; 
    }
    
    p {
      color: blue; 
    }

5. What does the !important expression mean in CSS?

CSS: 
    p {
      color: blue !important;
    }
    
    p {
      color: red;
    }
